Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header Label the top part of a page
div div Sets the page divisions apart
ol ordered list creates a numbered list of items in order
ul unordered list creates a list of items using bullet points in no specific order
li list item creates a list item
html html designates that the document includes html
head head designates the top of the document
body body designates the body of the document
title title indicates the title of the page
meta meta designates the character set of the page
br line break creates a new line in a document

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication<em> technical sophistication
strong strong make bold text <strong>technical sophistication<strong> technical sophistication
a anchor make a hyperlink <a href="https://learnenough.com/html">Learn Enough to be Dangerous/a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Display pieces of markup or source code <code>code</code> code
span span mark up part of a text or part of a document <span>Moby-Dick</span> Moby Dick
blockquote blockquote mark up a quote within the document <blockquote>This is an example of blockquote.</blockquote>
